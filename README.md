# 🌿 EcoAudit ASIR — Sustainability Calculator
**Jheremy Sánchez & Ayoub Sfili | ITB Group G11**

A data-driven web engine designed to analyze and project energy consumption for the **Institut Tecnològic de Barcelona**. This tool transforms historical data into actionable sustainability roadmaps using professional-grade frontend logic.

🌐 **Live Demo:** [View EcoAudit ASIR](https://jheremysanchez7e9-web.github.io/TA08_Calculadoraestalvienergetic/)

---

### 🚀 Key Features

* **Smart Projection Engine (RA 3.1a):** Automated forecasting for 8 critical metrics (Electricity, Water, Gas, and Consumables) for both annual and academic cycles.
* **Impact Reduction Roadmap (RA 3.1b):** Built-in logic to quantify a **30% reduction goal** over 3 years, applying Circular Economy principles.
* **Temporal & Seasonal Logic:** Advanced JS algorithms that model real-world fluctuations (winter heating peaks, summer cooling, and academic activity spikes).
* **Multi-Language UI:** Native support for English, Spanish, and Catalan via a custom i18n dictionary.
* **Visual Analytics:** Interactive dashboards powered by **Chart.js** for comparative resource tracking.
